# UFOs

## The purpose of this project was to create a webpage with a dynamic table covering UFO sitings. 

## Project Overview:
1. Build a site with javascript and html to include a title with a image background, body, and table.
2. Make the table dynamic by adding 5 filters to sort and view UFO sitings. 
3. Using bootstrap components and css to customize and make the the webpage more appealing and user friendly.

## Resources
- Software: HTML, CSS, Python 3.7.10, Conda 4.10.3, Jupyter Notebook 6.3.0, Visual Studio Code 1.60.2
